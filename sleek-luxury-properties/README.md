# Property Landing Page

A modern real estate landing page built with React, Tailwind CSS, and GSAP.  
This project demonstrates a clean UI, dark and light mode support, and responsive design for property and real estate platforms.

## Features

- React with Vite
- Tailwind CSS styling
- GSAP animations
- Dark mode and light mode
- Responsive layout
- Accessible UI
- SEO-friendly structure
